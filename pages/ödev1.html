Fonksiyon Kullanımı

Javascript kodu yazan bir çok developer this anahtar kelimesini kullanmıştır. Fakat C# gibi nesne tabanlı yazılım geliştiricilerin çoğu this anahtar kelimesini kullanırken kafasının karıştığını biliyorum. Aslında javascript teki this ile normal nesneye yönelimli programlamadaki this aynı değildir. Javascript nesnye yönelimli programlamaya karşılık fonksiyon tabanlı bir dildir. Fonksiyon çağırmalarında this anahtar kelimesi önemli bir rol oynar.

var result = { log:function(text){ document.getElementById('result-log').innerHTML = text; } }; function sum(a,b) { return a + b; } result.log(sum(2,3));

Yukarıda standart bir fonksiyon tanımladık. Bu fonksiyon ancak çağırıldığı(invoke) zaman yürütülür. sum(2,3) metodunu çağırdıktan sonra sonuç ekranında 5 yazdığını göreceksiniz. Peki bu fonksiyon herhangi bir nesneye acaba bağlandı mı? Javascript daima varsayılan olarak global bir nesneye sahiptir. Bu global nesne browser için window nesnesidir. Aslında yukarıda ki fonksiyon window global nesnesine bağlanmış oldu. Yani aşağıda görüldüğü gibi bu fonksiyonu window.sum(2, 3) olarak da çağırabiliriz.

function sum(a,b) { return a + b; } console.log(window.sum(2, 3)); // geriye 5 döndürecek.

İşte this anahtar kelimesinin sihri buradadır. Eğer bir fonksiyon çağrıldığı zaman kendi içinde(scope) hangi nesneye bağlanıyorsa o nesne this anahtar kelimesini işaret eder.

function newFunction() { return this; } console.log(newFunction()); // geriye window nesnesini geriye döndürür.

Yine başka bir örnek verelim. global bir değişken tanımlayalım.

var isGlobalWorld = true; function doCar() { var isGlobalWorld = false; //başına var koyarak isGlobalWorld nesnesi local hale getirildi. console.log(isGlobalWorld); //Geriye false döndürür. çünkü isGlobal değişkeni fonksiyon içinde local olarak tanımlandığı için. console.log(this.isGlobalWorld); //Geriye true döndürür. çünkü this anahtar kelimesi window nesnesini işaret eder. isGlobal değişkeni global tanımlandığı için(yani fonksiyon dışındaki isGlobal değişkeni) window nesnesine bağlanır. console.log(window.isGlobalWorld); } window.doCar();

Şimdi Bir Arabayı Javascript'de nesne kullanarak tanımlayalım

JavaScript değişkenlerinin veri değerleri için kapsayıcılar olduğunu zaten öğrendiniz. Bu kod, car adlı bir değişkene basit bir değer (Fiat) atar :

var car = "Fiat";

Nesneler de değişkenlerdir. Ancak nesneler birçok değer içerebilir. Bu kod, car adlı bir değişkene birçok değer (Fiat, 500, beyaz) atar :

var car = {type:"Fiat", model:"500", color:"white"};



Nesne Tanımı

Değişmez bir nesne içeren bir JavaScript nesnesi tanımlarsınız (ve oluşturursunuz):

var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

Boşluklar ve satır kesmeleri önemli değildir. Bir nesne tanımı birden fazla satıra yayılabilir:

var person = {
firstName: "John",
lastName: "Doe",
age: 50,
eyeColor: "blue"
};

Nesne Özelliklerine erişme

Nesne özelliklerine iki şekilde erişebilirsiniz:

objectName.propertyName

veya

objectName["propertyName"]
